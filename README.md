About the Project
-----------------

The final project of my second semester. A portfolio Website to see how good my HTML and CSS skills have become.

Project Goal
------------

The goal of this project was to create a website about us as developers, to have multiple pages and within my version a light and dark mode.

Features
- Pressing on the downloadbar will bring you to the corresponding repository.
- The social icons on the bottom left will bring you to my social media accounts.
- Pressing on the DM logo on the top right will switch to light mode or dark mode.

What I Practiced
----------------

- Creating a Website with multiple pages
- Adding a light and dark mode
- using css-variables and functions 

How To Download
---------------

1. Go to the top right of the project and download the code in a ZIP file.
2. Unzip the ZIP file.
3. Check that VS Code is installed.
4. Open the folder with VS code.
5. Ideally use Restclient extension to launch the Website on localhost.
